  <action>
Create `.github/workflows/` directory and `build-push.yml` workflow.

The workflow must:
1. Trigger on `push` to `main` branch only
2. Define `HARBOR_REGISTRY` and `HARBOR_PROJECT` as env vars (easy to change without touching job steps)
3. Run two jobs in PARALLEL: `build-backend` and `build-frontend`
4. Each job: checkout → login to Harbor → build+push image with two tags (sha + latest)
5. Use `docker/login-action`, `docker/build-push-action` (pinned to major version)

```yaml
name: Build and Push Docker Images

on:
  push:
    branches:
      - main

env:
  HARBOR_REGISTRY: harbor.hasslab.pro
  HARBOR_PROJECT: cell

jobs:
  build-backend:
    name: Build & Push Backend
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Log in to Harbor
        uses: docker/login-action@v3
        with:
          registry: ${{ env.HARBOR_REGISTRY }}
          username: ${{ secrets.HARBOR_USERNAME }}
          password: ${{ secrets.HARBOR_PASSWORD }}

      - name: Build and push backend image
        uses: docker/build-push-action@v6
        with:
          context: .
          file: Dockerfile.backend
          push: true
          tags: |
            ${{ env.HARBOR_REGISTRY }}/${{ env.HARBOR_PROJECT }}/cell-backend:${{ github.sha }}
            ${{ env.HARBOR_REGISTRY }}/${{ env.HARBOR_PROJECT }}/cell-backend:latest

  build-frontend:
    name: Build & Push Frontend
    runs-on: ubuntu-latest
    steps:
      - name: Checkout
        uses: actions/checkout@v4

      - name: Log in to Harbor
        uses: docker/login-action@v3
        with:
          registry: ${{ env.HARBOR_REGISTRY }}
          username: ${{ secrets.HARBOR_USERNAME }}
          password: ${{ secrets.HARBOR_PASSWORD }}

      - name: Build and push frontend image
        uses: docker/build-push-action@v6
        with:
          context: .
          file: Dockerfile.frontend
          push: true
          tags: |
            ${{ env.HARBOR_REGISTRY }}/${{ env.HARBOR_PROJECT }}/cell-frontend:${{ github.sha }}
            ${{ env.HARBOR_REGISTRY }}/${{ env.HARBOR_PROJECT }}/cell-frontend:latest
```

Note: `github.sha` is the full 40-char SHA. Both jobs run in parallel (no `needs:` dependency).
`HARBOR_PROJECT: cell` — change this env var if Harbor project name differs.
