Create Dockerfile.frontend — nginx:1.25-alpine serving frontend/ static files.

```dockerfile
FROM nginx:1.25-alpine

RUN rm -rf /usr/share/nginx/html/*

COPY frontend/ /usr/share/nginx/html/

RUN printf 'server {\n\
    listen 80;\n\
    server_name _;\n\
    root /usr/share/nginx/html;\n\
    index index.html;\n\
    location / {\n\
        try_files $uri $uri/ /index.html;\n\
    }\n\
}\n' > /etc/nginx/conf.d/default.conf

EXPOSE 80
```

nginx serves index.html for all routes (handles JS fetch calls to backend separately at runtime via env config or proxy — not wired here, this is a static artifact image).
